This is your first blog post! You can edit this content through the admin panel at `/admin`.

## Features

- **Static-first**: Built with Astro for optimal performance
- **Content Management**: Edit posts, images, and more through Decap CMS
- **Modern UI**: Beautiful components with Shadcn UI
- **Git-based**: All content is stored in your repository

## Getting Started

1. Visit `/admin` to access the content management panel
2. Create your first blog post
3. Customize the design to match your brand

Happy building!
